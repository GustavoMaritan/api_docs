  • POST Gerar
    Url
    http://localhost:3000/api/boleto/:banco?t
    Descrição
    Método responsavel por gerar boleto e retornar url para impressão ou download.
    Parâmetros
    Nome Tipo Opções Default Opcional
    banco String brasil,sicredi,sicoob Não
    t String print,download print Sim
    Opções Body [Parâmetro=banco]

    "dataImpressao":Date,//Data impressão do boleto(dd/MM/yyyy)
    "dataGeracaoBoleto":Date,
    "numeroDocumento":String,

    "carteira":String,
    "numeroConvenio":Number,
    "pagador":{
    "nome":String,//Formato: xxxxx xxxxx xxxxx
    "edereco":String,
    "cep":String,
    "cpfCnpj":String

    "nome":String,

    },

    "nossoNumero":String
    }
