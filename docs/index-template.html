    Url
    http://localhost:11002/api/boleto/:banco?t
    Descrição
    Método responsavel por gerar boleto e retornar url para impressão ou download.
    Parâmetros
    Nome Tipo Opções Default Obrigatório
    banco String brasil, sicredi, sicoob Sim
    t String print, download print Não
    Opções Body [Parâmetro=banco]
    Brasil
    {
    dataImpressao: { type: Date, descricao: 'Data impressão do boleto' },
    dataGeracaoBoleto: Date,
    numeroDocumento: String,
    especie: String,
    carteira: String,
    numeroConvenio: String,
    pagador: {
        nome: String,
        edereco: String,
        cep: String,
        cpfCnpj: String,
    },
    beneficiario: {
        nome: String,
        cpfCnpj: String,
        codigo: String,
        codigoDigito: String,
        agencia: String,
        agenciaDigito: String
    },
    parcelas: [{
        numero: Number,
        valor: Number,
        vencimento: Date,
        nossoNumero: String
    }]
}
    Retorno OK

        type: Array,
