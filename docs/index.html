  • POST Gerar
    Url
    http://localhost:3000/api/boleto/:banco?t
    Descrição
    Método responsavel por gerar boleto e retornar url para impressão ou download.
    Parâmetros
    Nome Tipo Opções Default Opcional
    banco String brasil,sicredi,sicoob Não
    t String print,download print Sim
